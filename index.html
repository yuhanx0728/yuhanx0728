DeepNightmare

An Experiment between Art and ML

DeepNightmare is a project asking how scary AI can be. The DeepDream algorithm was implemented using a ResNet architecture trained on a customized dataset of 3K “scary”/”not-scary” images, to create vivid horror images. My main responsibility in this project included integrating a GUI for users to interact with trained images in real time, and collecting data.
